| الجدول               | وظيفته                                   |
| -------------------- | ---------------------------------------- |
| `tenants`            | الشركات المسجلة في النظام                |
| `users`              | مستخدمو كل شركة، وأول مستخدم سيكون Admin |
| `accounts`           | دليل الحسابات لكل شركة                   |
| `subscription_plans` | خطط الاشتراك                             |
| `customers`          | عملاء الشركة                             |
| `subscriptions`      | ربط العميل بخطة                          |
| `invoices`           | الفواتير الشهرية                         |
| `payments`           | المدفوعات على الفواتير                   |
| `journal_entries`    | رأس القيد المحاسبي                       |
| `journal_lines`      | أطراف القيد: Debit / Credit              |
